## 知识栏内容示例

### 美食卡

| 卡名 | 知识栏文字 |
|------|-----------|
| 北京烤鸭 | "挂炉烤鸭用枣木或梨木，果木燃烧温度稳定且带甜香。鸭坯挂炉前要往体内灌开水撑皮，烤时皮下脂肪融化外溢，形成酥脆表皮。吃法分三种：蘸糖吃皮、卷饼卷葱吃肉、鸭架熬汤。片鸭要求每片带皮带肉，厚薄均匀。" |
| 炸酱面 | "炸酱用黄酱和甜面酱按比例混合，猪肉丁小火慢炒出油，这叫'小碗干炸'。面码至少八样：黄瓜丝、心里美萝卜丝、豆芽、水萝卜、青蒜、芹菜丁、白菜丝、黄豆。夏天吃过水面降温，冬天吃锅挑面（不过水）保热。面码的作用是提供不同口感和维生素。" |
| 豆汁焦圈 | "豆汁是绿豆制粉丝时的副产品——淀粉沉淀后上层液体经乳酸菌发酵而成，含丰富B族维生素和蛋白质。酸涩口感来自发酵产生的有机酸。焦圈是明矾和面炸成的环形面食，酥脆中空，油脂能中和豆汁的酸感。传统搭配还有辣咸菜丝。" |
| 涮羊肉 | "正宗涮羊肉用铜锅炭火，清水只放葱段、姜片、枸杞。手切鲜羊肉取自大尾巴羊的上脑、磨裆、黄瓜条部位，逆纹切薄片，保留适量脂肪才有口感。肉片入沸水变色即捞——时间过长蛋白质过度收缩会变硬。蘸料以芝麻酱为主，配腐乳汁、韭菜花、辣椒油。" |
| 东坡肉 | "苏轼在黄州写《猪肉颂》记录做法：'净洗铛，少著水，柴头罨烟焰不起。待他自熟莫催他，火候足时他自美。'关键工艺是密封慢炖——砂锅加黄酒、酱油、冰糖，小火焖两小时以上，胶原蛋白充分水解使肉质软糯。杭州做法要求五花肉切五厘米见方块，一块即一份。" |
| 龙井虾仁 | "选用清明到谷雨期间的明前龙井和太湖白虾仁。虾仁用蛋清、淀粉上浆锁住水分，温油滑至玉白色捞出。茶叶先用少量热水泡开，最后和虾仁同炒不超过十秒。成菜要求虾仁洁白鲜嫩、茶叶碧绿清香，整道菜不放酱油不放糖，全靠食材本味。" |
| 西湖醋鱼 | "选用一斤半左右的草鱼，宰杀前需饿养一到两天以排净泥腥。整鱼不切断，用沸水汆熟——火候是关键，过火肉就柴。糖醋卤汁以米醋、白糖、酱油、湿淀粉调制，趁鱼刚出锅热度浇上，利用余温使汁液半凝挂住鱼身。全程不加油，是极少数无油烹饪的名菜。" |
| 松鼠桂鱼 | "选用鳜鱼去头骨、脊骨，保留鱼肉连皮不断。改菱形花刀后拍淀粉入高温油炸——鱼肉受热外翻呈松果状，尾部翘起形似松鼠。关键是花刀深度：切到鱼皮但不切断皮。糖醋汁用番茄酱、白醋、白糖熬制，趁热浇淋，热汁遇炸物产生嗞嗞声响。" |
| 麻婆豆腐 | "清同治年间成都万福桥边陈兴盛饭铺首创。用嫩豆腐切块焯水，牛肉末炒酥，郫县豆瓣酱炒出红油，加花椒面和辣椒面同烹。出锅时撒现磨花椒粉——'麻'字是点睛。业内总结其风味为八字：麻、辣、烫、香、酥、嫩、鲜、活。如今是川菜中传播最广的单品，被列入日本国民料理。" |
| 重庆火锅 | "锅底以牛油为主料，加入豆瓣酱、干辣椒、花椒、草果、八角等二十余种香料熬制。牛油熔点高，冷却后凝固可反复使用。涮毛肚有讲究：鲜毛肚须当天现发，下锅后快速抖散，变色卷曲立即捞出——口感脆弹，过熟则韧。鸭肠、黄喉同理，都是考验'秒数'的食材。" |
| 白切鸡 | "选三黄鸡（黄嘴、黄脚、黄毛），整鸡浸入微沸水中'虾眼水'状态（约90度）慢浸至熟，再入冰水浸泡收紧表皮。这样做出的鸡皮脆肉嫩，骨边微带血丝说明蛋白质未过度变性，保留了最多肉汁。蘸料只用姜蓉葱油或沙姜酱油。粤菜衡量厨师水平，第一关就是白切鸡。" |
| 虾饺 | "虾饺皮用澄面（小麦淀粉）加少量生粉开水烫制，趁热揉成面团后擀薄。澄面受热后变半透明，蒸熟后能看见馅料颜色。馅用鲜虾仁整只或切粒，加少量肥膘和笋丁增香添脆。标准虾饺要求有十三道褶，形如弯梳，大小刚好一口一个。" |
| 佛跳墙 | "福建名菜，正式创制于清光绪年间福州聚春园菜馆。以鲍鱼、海参、鱼翅、干贝、花胶等十余种干货为主料，先分别发制处理，再层层码入绍兴酒坛，加高汤文火煨制十余小时。荤而不腻的关键在于各原料错开火候入坛，且陈年花雕酒有去腥增香的化学反应。" |
| 热干面 | "关键工序是'掸面'：碱水面煮至八分熟捞出，趁热拌入香油摊凉晾干——这步使面条表面形成油膜，防粘且增加弹性。食用时过沸水烫十秒捞出，加芝麻酱（纯芝麻研磨）、香油、生抽、醋、辣椒油、酸豆角、葱花拌匀。芝麻酱遇热会变稀，裹面效果最好的时间只有搅拌后的前两分钟。" |
| 臭鳜鱼 | "徽菜代表。新鲜鳜鱼用盐抹匀后层层码放在木桶中，上压重物，在室温下发酵六到七天。乳酸菌分解蛋白质产生氨基酸（提鲜）和硫化物（产臭味）。烹饪时用红烧法——高温可分解部分异味物质，同时氨基酸带来极强鲜味。原理类似奶酪和纳豆的发酵。" |
| 盐水鸭 | "南京人年消费鸭子超过一亿只。盐水鸭的工序：先用花椒盐反复搓揉腌渍，再入老卤水中低温浸煮（水温保持在85度左右，不能沸腾）。低温慢煮保证肉质细嫩、汁水不流失。成品皮白肉粉骨微绿（骨髓中铁质被盐析出氧化所致）。秋季用桂花盐腌制的叫'桂花鸭'。" |
| 剁椒鱼头 | "湖南家庭每年秋季大量制作剁椒：新鲜红辣椒剁碎加盐密封发酵，坛口用水封隔绝空气，乳酸发酵产生酸香。花鲢鱼头对半劈开铺上剁椒，大火蒸十二分钟。鱼头胶原蛋白丰富、肉质嫩滑，高温蒸制使辣椒中的辣椒素渗入鱼肉。盘底汤汁是辣油、鱼汁和蒸汽凝液的混合物，浇饭极香。" |
| 过桥米线 | "云南蒙自的吃法。关键是那碗滚烫的鸡汤——表面浮着一层厚厚的鸡油，隔绝空气使汤温保持在90度以上而不冒热气。食客依序放入生肉片、鹌鹑蛋、豆皮、韭菜等配料，利用汤温将食材烫熟。放料顺序按食材熟成时间排列：肉片先下，蔬菜后放，最后下米线。" |
| 肉夹馍 | "白吉馍是发面团贴在吊炉内壁高温烤制，烤好的馍外壳焦脆、内部起层——行话形容为'铁圈虎背菊花心'。腊汁肉用花椒、八角、桂皮、草果等大料卤制十小时以上，肉中胶原蛋白完全水解成明胶，所以冷了会凝冻。夹馍时热馍剖开夹入热肉，面饼吸收肉汁后内外口感层次分明。" |
| 羊肉泡馍 | "死面烙馍（不发酵）由食客自己手掰成黄豆大小的碎块——颗粒大小直接影响煮后口感。后厨收到碗后在滚沸的羊骨老汤中煮制，馍粒吸收汤汁膨胀变软但不糊。汤底由羊棒骨小火熬制十二小时以上，胶原蛋白溶出使汤体浓白。配糖蒜和辣子酱解腻。" |
| 驴肉黄面 | "敦煌特色面食。黄面的黄色来自蓬灰水（碱性矿物溶液）与面粉中类黄酮反应后的显色。面条拉成韭叶宽，口感爽滑筋道。驴肉脂肪含量低于牛羊肉，蛋白质含量高，切丁后用酱油、香料煨制成浇头。敦煌地处河西走廊绿洲，自古是丝路补给站，饮食融合了中原和西域风格。" |
| 桂林米粉 | "米粉以早稻籼米为原料，浸泡后磨浆、蒸熟、压榨成圆条状。桂林吃法分卤粉和汤粉，本地人偏爱干捞卤粉：粉不加汤，拌以猪骨、牛骨、草果、八角等熬制的卤水，配料有锅烧（油炸五花肉）、花生米、酸笋、酸豆角。酸笋提供乳酸发酵的酸味，与肉香形成对比。" |
| 洛阳水席 | "唐代起源的宴席形式，共二十四道菜，每道均带汤或汁水，且流水般逐道上桌——'水'字一指汤多，二指上菜不断。首道'牡丹燕菜'以白萝卜为主料切极细丝，配以高汤和蛋丝，摆成牡丹花造型。这道菜体现了中国烹饪'以素托荤'的技法——用廉价食材模拟名贵口感。" |
| 灌汤包 | "馅心拌入猪皮冻（猪皮熬制的明胶冷却凝固后切碎），常温下是固态。入蒸笼后温度升高，明胶融化为液态汤汁，被面皮包裹其中。皮面用冷水和面，不放酵母——死面皮薄且韧，承托汤汁不破。正确吃法：先咬小口释放蒸汽，吸汤，再吃馅和皮。烫伤事故多因跳过第一步。" |
| 糖醋鲤鱼 | "鲁菜经典。选用黄河鲤鱼——黄河水含沙量高、水温低，鱼生长缓慢所以肉质紧实。'牡丹花刀'是在鱼身两侧切交叉斜刀至骨不断——炸制时肌纤维受热收缩使鱼身弓起、花刀张开。糖醋汁以白醋、白糖、番茄酱调制，趁鱼炸好瞬间浇上，利用温差使汁液迅速挂壁。" |
| 酥油茶 | "藏区传统饮品。砖茶（发酵后压制的紧压茶）熬煮出浓茶汤，倒入'甲董'（木制长筒）中加入牦牛酥油和盐巴，反复抽提搅打使油脂乳化——原理类似打奶泡。每100毫升酥油茶约含400千焦热量，高原地区日均气温低、氧含量不足平原的60%，高热量饮品是生理刚需。" |
| 潮汕牛肉火锅 | "核心在'鲜'——牛从屠宰到上桌控制在四小时内，肉温不经冷链。一头牛按肌肉纹理和脂肪分布分为十余个部位：吊龙（腰脊肉，脂肪纹均匀）、匙仁（肩胛嫩肉）、五花趾（牛蹄筋附近胶质多）、胸口朥（纯脂肪，涮后酥脆）。手切确保纤维完整，冷冻会破坏细胞壁使口感失弹。蘸沙茶酱食用。" |
| 工夫茶 | "潮汕传统泡茶法，'工夫'指技巧和讲究。用朱泥小壶（约100毫升）配三只薄瓷杯，茶叶用凤凰单丛或铁观音。冲泡要'高冲低斟'：注水从高处使茶叶翻滚充分浸出，倒茶时壶嘴贴杯低斟减少香气散失。'关公巡城'是巡回注茶使浓度均匀，'韩信点兵'是末尾几滴精华平均分配。" |

### 景区卡

| 卡名 | 知识栏文字 |
|------|-----------|
| 故宫 | "占地72万平方米，是世界现存最大的木结构宫殿建筑群。地面铺的'金砖'实为苏州御窑烧制的细泥方砖——从取土到成砖需两年，成品敲之有金属声。宫殿采用榫卯结构不用一颗铁钉，屋顶琉璃瓦的黄色来自铅锑釉在高温下的化学反应。整座宫城沿中轴线对称布局，体现'居中为尊'的建筑哲学。" |
| 长城·八达岭 | "明长城总长8851公里，是人类历史上最大规模的军事工程。城墙依山势起伏修建，利用地形形成制高点。烽火台间距约5公里（可视距离极限），通过烟火信号传递军情，速度超过骑兵。墙体内部用夯土填充，外包条石或城砖，宽度允许五马并行。八达岭段海拔约600米，坡度最陡处达40度。" |
| 兵马俑 | "1974年西杨村农民打井时发现。三座坑共出土陶俑约八千件，按真人比例制作（身高1.75-1.97米）。面部特征各异，推测采用模块化生产：身体用模具批量制作，面部再由工匠逐一手工修饰。俑身背部刻有工匠姓名——这是秦代'物勒工名'的质量追溯制度。出土时残留矿物质颜料痕迹（朱砂、石绿、石青），证明原本为彩绘。" |
| 西湖 | "原为潟湖——海湾被泥沙封闭后形成。面积6.39平方公里，平均水深1.5米。唐代白居易任杭州刺史时主持修筑白堤疏浚湖水，宋代苏轼任知州时再次疏浚并筑苏堤。两次大规模水利工程奠定了西湖今日的格局。湖水主要由周边山溪补给，每年换水一次以维持水质。2011年列入世界文化遗产。" |
| 莫高窟 | "始凿于公元366年，历经十六国、北魏、隋、唐至元代，持续营造约一千年。现存洞窟735个、壁画4.5万平方米、彩塑2400余身。壁画颜料为矿物质（朱砂、石青、石绿、金箔），化学性质稳定所以历千年不褪。1900年道士王圆箓发现藏经洞，内藏5万余件文献，后大量被斯坦因、伯希和等外国探险家廉价购走。" |
| 布达拉宫 | "建于海拔3700米的红山上，公元7世纪松赞干布始建，17世纪五世达赖重建为现今规模。建筑分白宫（行政）和红宫（宗教），共13层、房间超过一千间。墙体以花岗石为基础，上层用夯土和木结构。外墙涂白用的涂料含石灰、牛奶和冰糖——牛奶中的酪蛋白起粘合和防水作用。每年藏历9月粉刷一次。" |
| 拙政园 | "建于明正德年间，占地5.2公顷，是苏州现存最大的古典园林。设计遵循'虽由人作，宛自天开'的造园原则——利用水面反射扩大空间感，假山障景制造移步换景效果。园中水面占总面积近三分之一，通过桥、廊、亭将空间分隔又连通。植物配置四季有景：春兰、夏荷、秋���、冬梅。1997年列入世界文化遗产。" |
| 漓江 | "漓江两岸为典型喀斯特地貌——石灰岩在碳酸水溶蚀作用下形成峰林、溶洞和地下河。水质清澈是因为石灰岩过滤了大部分悬浮物。桂林至阳朔河段83公里是景观最集中的区域。20元人民币背面图案取自阳朔兴坪段'黄布倒影'。鸬鹚捕鱼是当地延续数百年的传统渔法——鸟颈部扎环使其吞不下大鱼。" |
| 龙门石窟 | "始凿于北魏孝文帝迁都洛阳（公元494年），历经东魏、北齐、隋、唐四百余年营造。现存洞窟2345个、佛像十万余尊。最大的卢舍那大佛通高17.14米，雕凿于唐高宗时期——面容圆润、嘴角微扬，体现唐代审美。石材为石灰岩，质地细腻利于精雕，但也易风化。近代大量佛首被盗凿贩卖至海外。" |

| 景德镇瓷器 | "青花瓷的蓝色来自钴矿颜料——元代从波斯进口'苏麻离青'，明代改用国产'平等青'和'回青'，不同产地的钴矿呈色各异。制瓷需经72道工序：从采石碎料、淘洗制泥、拉坯修坯，到施釉、绘画、入窑烧制（1300度高温）。景德镇瓷土含高岭土——这种矿物以景德镇高岭村命名，是制瓷的关键原料。" |
| 敦煌壁画（飞天） | "飞天形象在莫高窟中从北凉延续到元代，造型随时代变迁：早期受印度犍陀罗风格影响，体态壮硕、飘带短小；到盛唐变为S形曲线身姿、长飘带环绕。飞天不借助翅膀或云朵——纯粹靠飘带和身体姿态表现飞行感，这在世界美术史上是独特的。壁画颜料用朱砂（红）、石青（蓝）、石绿（绿）等矿物质，稳定性远超植物颜料。" |
| 龙泉青瓷 | "'梅子青'釉色的形成原理：釉中含铁量约1%-3%，在还原气氛（窑内缺氧）下，三价铁还原为二价铁呈现青绿色。需多次上釉多次素烧，釉层厚度达到普通瓷器的三倍以上，才能产生如玉般的质感。烧制温度控制在1250-1280度之间，波动超过10度就会偏色。南宋时龙泉青瓷经海上丝路出口至东南亚、中东和东非。" |
| 宜兴紫砂 | "紫砂泥是宜兴丁蜀镇特有的含铁质粘土矿，分紫泥、红泥、绿泥三大类。烧结后泥料中形成双气孔结构（开放气孔率约5%）——既透气又不漏水，这是紫砂区别于其他陶瓷的物理特性。长期使用后茶多酚和茶油渗入气孔壁形成'包浆'，即所谓'养壶'。不同泥料适配不同茶类：紫泥适合普洱、红泥适合乌龙。" |
| 苏绣 | "苏绣的核心技法'劈丝'：一根桑蚕丝线直径约0.02毫米，绣娘将其分劈为2丝、4丝直至128分之一丝。极细丝线用于绣制眼珠、水波等需要渐变效果的部位。'乱针绣'是近代革新技法——打破传统排针方式，用交叉长短针模拟油画笔触效果，适合绣制光影复杂的人物肖像和风景。" |
| 景泰蓝 | "学名铜胎掐丝珐琅，工艺流程：先锻造铜胎→用扁铜丝掐制图案轮廓焊接于胎体→将石英砂、硼砂和金属氧化物调制的珐琅釉料填入铜丝格中→入窑800度烧结→打磨整平→镀金。蓝色釉料中的呈色剂是氧化钴，绿色为氧化铜，红色为氧化铁。全套工序需要制胎、掐丝、填釉、烧蓝、打磨、镀金六个工种协作完成。" |
