# 山河游记

> 中国高铁旅行桌游 · 2 人对战 · 12 回合 · 30 分钟一局

**▶ 在线试玩：[yingwang.github.io/shanhe-youji](https://yingwang.github.io/shanhe-youji/)**

两位旅行者用 12 天坐高铁游遍中国。规划路线、抢热门车票、打卡 5A 景区、品尝八大菜系、收集文化瑰宝。体力、金钱、时间三种资源各卡一头，每回合都得做取舍。
